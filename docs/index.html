About QuickChat

QuickChat is a real-time file sharing and messaging application built with modern web technologies:

  • Node.js server with Express framework
  • Socket.IO for real-time communication
  • SQLite database for storing room and file information
  • Vite + TypeScript frontend

This GitHub Pages site provides information about the project. To use QuickChat, you need to run the server locally or deploy it to a Node.js hosting service.
